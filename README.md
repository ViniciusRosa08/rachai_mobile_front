# 🚗 Rachai - Aplicativo de Carona solidaria

Aplicativo Android para locação de veículos entre usuários, desenvolvido em Java utilizando Android Studio.

## 📱 Funcionalidades

- Cadastro e login com validações
- Listagem de veículos disponíveis
- Criação e gerenciamento de locações
- Área do cliente e do motorista
- Avaliações e histórico de viagens

## 🛠️ Tecnologias Utilizadas

- Java
- Android Studio
- Firebase (autenticação, se aplicável)
- Azure MySQL (banco de dados - planejado)
- Git e GitHub
